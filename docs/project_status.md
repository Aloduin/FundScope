# FundScope 项目状态

> 最后更新：2026-03-22
> 当前阶段：Phase 3 — 策略能力增强
> 项目目标：构建一个面向个人投资者的基金研究、组合诊断、策略验证与模拟投资系统，实现”数据驱动 + 策略可验证 + 决策可解释”。

---

## 1. 项目概览

### 1.1 项目定位
FundScope 是一个基金投资辅助系统，聚焦以下核心能力：

- 基金研究：基金信息、指标计算、多维度评分、赛道分类、净值历史分析
- 组合诊断：持仓导入、集中度分析、赛道重叠检测、优化建议
- 策略验证：定投、均线择时、动量轮动等策略回测
- 模拟投资：虚拟账户、买卖记录、收益统计
- 数据基础设施：真实 akshare 数据接入、缓存、存储、数据校验

### 1.2 系统边界
当前系统不包含：

- 实盘交易对接
- 自动下单
- 多用户系统
- 云端部署
- 高频交易策略
