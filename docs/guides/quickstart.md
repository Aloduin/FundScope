# FundScope 快速开始指南

本指南帮助你快速上手 FundScope 基金投资辅助系统。

## 1. 安装

### 1.1 前置要求

- Python 3.13 或更高版本
- uv（推荐）或 pip

### 1.2 克隆仓库

```bash
git clone https://github.com/Aloduin/FundScope.git
cd FundScope
```

### 1.3 安装依赖

**使用 uv（推荐）**

```bash
# 安装 uv（如果尚未安装）
pip install uv

# 同步依赖
uv sync
```

**使用 pip**

```bash
pip install -e ".[dev]"
```

---

## 2. 启动应用

```bash
uv run streamlit run ui/app.py
```

应用将在 `http://localhost:8501` 启动。

## 3. 使用指南

### 3.1 基金研究（页面 1）

**功能：** 搜索基金、查看评分、赛道分类

**步骤：**
1. 在左侧边栏点击 "📈 基金研究"
2. 输入基金代码（如 `000001`）
3. 点击 "搜索"
4. 查看基金信息、评分、指标

**输出：**
- 基金基本信息（名称、类型、经理、规模）
- 赛道分类（主赛道、全部标签）
- 六维度评分（收益、风险、稳定性、成本、规模、经理）
- 绩效指标（年化收益、最大回撤、波动率、夏普、胜率）

### 3.2 持仓诊断（页面 2）

**功能：** 录入持仓、分析集中度、获取建议

**步骤：**
1. 在左侧边栏点击 "📊 持仓诊断"
2. 输入基金代码、名称、持仓金额
3. 点击 "添加"
4. 重复步骤 2-3 添加多只基金
5. 点击 "开始诊断"

**输出：**
- 集中度风险（HHI 指数）
- 有效持仓数（effective_n）
- 赛道重叠检测
- 防守资产检测
- 优化建议

**示例：**

| 基金代码 | 基金名称 | 金额 | 权重 |
|----------|----------|------|------|
| 000001 | 基金 A | 10000 | 33.3% |
| 000002 | 基金 B | 20000 | 66.7% |

诊断结果：
- HHI: 0.555（中度集中）
- effective_n: 1.8（分散度不足）
- 建议：增加持仓数量或均衡配置

### 3.3 策略验证中心（页面 3）

**功能：** 创建虚拟账户、模拟交易、跟踪持仓

**步骤：**
1. 在左侧边栏点击 "🧪 策略验证中心"
2. 输入账户 ID（如 `my_account_001`）
3. 设置初始资金（如 `100000` 元）
4. 点击 "创建账户"
5. 在 "买入" 标签页输入基金信息和金额
6. 点击 "买入" 执行交易
7. 在 "卖出" 标签页执行卖出操作

**输出：**
- 账户摘要（现金、已用资金、持仓数量）
- 当前持仓列表
- 交易记录

## 4. 常见问题

### Q1: 基金数据是真实的吗？

MVP 阶段使用 mock 数据。第二阶段将接入真实 akshare 数据。

### Q2: 如何修改评分权重？

编辑 `shared/config.py` 中的 `SCORE_WEIGHTS_BY_TYPE` 字典。

### Q3: 如何添加新基金类型？

1. 在 `shared/config.py` 添加 `FUND_TYPE_MAPPING` 映射
2. 在 `SCORE_WEIGHTS_BY_TYPE` 添加对应权重

### Q4: 数据存储在何处？

- 时序净值：`data/parquet/<fund_code>.parquet`
- 业务数据：`data/sqlite/fundscope.db`

### Q5: 如何清空测试数据？

删除 `data/` 目录并重启应用。
